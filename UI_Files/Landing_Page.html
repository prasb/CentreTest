Welcome to CentreTest


Automation Testing As a Service [AtaaS].


The CentreTest Quick Start program incorporates automation experience, expertise, and best-in-class technology to tackle traditional automation challenges and deliver immediate ROI


Our unique cloud based model of Mobile Test Automation decreases your time to market, increases test coverage, and saves you money. Our activities include

  • Small & lean tests to lower maintenance
  • Loose coupling of tests & data
  • Periodic reviews to ensure test suites are green
  • Analyzing opportunities for in-sprint and delayed automation
  • Supporting execution of multiple regression tests during each sprint
  • Maintenance of an automation backlog and prioritize candidate test cases


The other key phases of a test automation project are:


  • Test Automation Framework design and Coding
  • Test Case Reorganization/Management
  • Test Automation Maintenance
